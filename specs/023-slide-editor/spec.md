# Spezifikation: Slide Editor (Spec 023)

## Ziel
Ein Editor für Slides, der das Erstellen, Bearbeiten und Speichern von Präsentationsfolien im Browser ermöglicht.

## Anforderungen
- Slides können erstellt, bearbeitet, gelöscht und gespeichert werden
- Unterstützung für Text, Bilder und einfache Formatierungen
- Speicherung im Backend (z.B. als JSON oder Markdown)
- Undo/Redo-Funktionalität
- Vorschau-Modus

## Akzeptanzkriterien
- Nutzer kann neue Slides anlegen und bestehende bearbeiten
- Änderungen werden persistent gespeichert
- UI ist responsiv und intuitiv

## Nicht-Ziele
- Kein Export als PowerPoint oder PDF
- Keine Animationen
