#### New Field
```prisma
model Course {
  // ... existing fields ...
  
  // NEW: Hero video for course detail page
  heroVideoPlaybackId String? @map("hero_video_playback_id")
  
  // ... relations ...
}
```

### Field Specification

| Field | Type | Required | Default | DB Column |
|-------|------|----------|---------|-----------|
| `heroVideoPlaybackId` | String | No | null | `hero_video_playback_id` |

**Description**: Mux Playback ID for the course hero video. When null, the course detail page displays a static hero image using the existing `imageDetail` field.

**Example Values**:
- `"xyw0xyx00D02TUYCpZjG6aKnHqI2tYTG00"` (Mux test asset)
- `null` (fallback to static image)
